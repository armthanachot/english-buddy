# English Buddy

### Plan
เป็น platform ในการเรียนรู้ภาษาอังกฤษแบบครบวงจร สามารถพัฒนาความรู้ และใช้งานได้จริง โดยมี feature หลักๆดังนี้ 
1. แปลภาษาอังกฤษ
2. การยกตัวอย่างวิธีการทำไปใช้จริง ในหลากหลายสถานการณ์ พร้อมรูปภาพประกอบ รวมถึงคำบรรยายแบบออกเสียง (สำหรับ free subscription จะได้วันละ x ครั้ง)
3. การแปลบทความ ทั้งบทความ (premium subscription)
    - auto detect คำศัพท์ที่น่าสนใจ ด้วย AI
    - สามารถ highlight คำที่ตนเองสนใจได้ แบบ manual
    - แต่ละคำที่ highlight หรือ AI detect ออกมาได้ ก็จะมีการนำไปยกตัวอย่างการนำไปใช้จริง แต่ละ situation
4. ในแต่ละสถานการณ์ที่ AI ยกตัวอย่างมาให้ สามารถโต้ตอบกับ chatbot เพื่อสอบถามเพิ่มเติมได้
5. ในอนาคต อาจจะเปิดเป็น tutorial platform โดยแบ่งเป็น `เรียนกับคุณครู AI` หรือ เรียนกับ `ผู้สอนแบบเป็นคน (มีทั้งคนไทย และเจ้าของภาษา)`
6. ต้องเก็บประวัติของการแปลในส่วนที่เป็น situation และรูปภาพ เอาไว้ เพื่อดึงออกมาแสดง และเป็นการลด token แต่หากมีการกด refresh ก็จะเคลียร์ของเดิมทิ้งไป หากต้องการกู้ของเดิมคืน อาจจะต้อง (premium subscription)
7. บริการแปลภาษาอังกฤษ เป็น ภาษาต่างๆ คิดเรทราคาแตกต่างกันออกไป (ขึ้นอยู่กับปริมาณ และภาษา output) พร้อมตราประทับ
8. การสร้าง quiz สำหรับภาษาอังกฤษ โดยจะนำประวัติของการใช้งานของ user คนนั้นๆ มาสร้างเป็น quiz เพื่อทดสอบความสามารถ
    - ความถี่ในการออก quiz เช่น user ใช้งานไปแล้ว 10 ครั้ง ก็ออก quiz มา
    - หรืออาจจะเป็น quiz ย่อยๆ ของการใช้งานแต่ละครั้งก็ได้
9. pre test แล้ว summarize ว่า user คนนั้นๆ ต้องพัฒนาส่วนไหน เป็น summarize แบบละเอียด
10. การดึงโจทย์ พร้อมเฉลย ของพวกข้อสอบ อย่าง toeic ปีเก่าๆมา แล้วเอามาอธิบายว่า ทำไมแต่ละข้อถึงตอบแบบนั้น
